# 手动验证指南 (Manual Verification Guide)

本文档用于验证 PySide6 版本与原 tkinter 版本的功能一致性。

---

## 前置条件

1. 安装 PySide6：
   ```bash
   .venv\Scripts\pip install PySide6
   ```

2. 确保以下库已安装（原项目依赖）：
   - pyvisa
   - pywin32
   - xlwings

3. 连接泰克示波器（MSO4/5/6、DPO7000 或 DPO5000 系列）

---

## 启动新版本

```bash
cd E:\Project\tools\MSO TOOL\EE-自动化测试\EE_Power_on_AutoTool_VER1.0
.venv\Scripts\python main.py
```

## 常见问题排查

### 连接示波器失败
- 检查 USB/GPIB 线缆是否连接
- 检查 NI-VISA 驱动是否安装
- 查看控制台日志中的设备信息

### Excel 操作失败
- 确保文件未被其他程序打开
- 检查文件路径是否包含中文/特殊字符

### 截图保存失败
- 确保示波器已连接
- 检查图片保存路径是否有写入权限
- 查看控制台 SCPI 命令执行日志
